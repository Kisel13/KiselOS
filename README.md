# KiselOS
KiselOS is a simple Computercraft operating system

I will be very happy if someone wants to help with the development! Please write to Telegram @Kisel130

**Main functions**

 - (Work In Progress) Quick installer
 - Run on the computer screen, or auto monitor search
 - Running programs from the /Programs/ folder
 - Opening Shell, or Lua console
 - Built-in adapted version of [MineExplorerPlus](https://github.com/Missooni/MineExplorerPlus) by [Missooni](https://github.com/Missooni)
