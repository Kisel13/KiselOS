# KiselOS
KiselOS это простая графическая оболочка для Computercraft.

Буду очень рад если кто-то захочет помочь с разработкой! Пишите в Telegram @Kisel130

**Основные функции**

 - Быстрый установщик
 - Запуск на экране компьютера, или автопоиск монитора
 - Запуск программ из папки /Programs/
 - Открытие Shell, или Lua консоли
 - Встроенная адаптированная версия [MineExplorerPlus](https://github.com/Missooni/MineExplorerPlus) созданного [Missooni](https://github.com/Missooni)
